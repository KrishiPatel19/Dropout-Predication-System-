# 🎓 Student Dropout Analysis & Prediction System

## 📌 Project Overview

This project focuses on **Student Dropout Analysis and Prediction** using multiple Data Mining, Machine Learning, Text Mining, and Web Mining techniques. The goal of the project is to analyze student academic data, identify factors contributing to student dropout, and build predictive models that can help educational institutions take preventive actions.

The repository contains:

* Data preprocessing and exploratory data analysis
* Machine learning models for dropout prediction
* WEKA implementation files
* Text Mining experiments
* Web Mining experiments
* Jupyter notebooks for model training and analysis
* Project documentation and write-up

# 📊 Dataset Information

The project uses a student academic dataset for dropout prediction.

### Dataset Characteristics

* Total Records: **4424**
* Total Features: **37**
* Target Variable: **Student Dropout Status**

The dataset is analyzed to understand patterns related to:

* Academic performance
* Student demographics
* Enrollment details
* Educational background
* Financial and social factors

## 1️⃣ Clone the Repository

```bash

```

## 2️⃣ Navigate to Project Folder

```bash

```

```bash

```

## 4️⃣ Run Jupyter Notebook
